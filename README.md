 WebDriverIO Setup Guide
 
This project uses WebDriverIO for API automation . Below is a step-by-step guide to set up the project and execute the test scenarios.

Prerequisites
Before running the tests, ensure that the following are installed on your machine:

- Node.js (LTS version recommended)
- npm (comes with Node.js)
- WebDriverIO (this project uses WebDriverIO for automation)


Running the Tests
- npm install
- npx wdio run wdio.conf.js
